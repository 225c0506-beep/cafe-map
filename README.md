# Cafe Map - 新宿エリアのカフェマップアプリ

Leaflet と Supabase を使ったカフェマップアプリです。新宿エリアのカフェを地図上で探したり、登録したり、タグで絞り込んだりできます。

## 機能

- **カフェの閲覧**: 地図上のピンまたはリストからカフェを探せる
- **カフェの登録**: 地図をクリックして位置指定＋写真アップロード
- **ユーザー認証**: メール＋パスワードでのログイン/新規登録
- **いいね**: 気に入ったカフェにいいね可能（1ユーザー1件）
- **タグ付け**: 10種類のタグでカフェをラベリング（気分/機能別）
- **タグフィルター**: AND/OR 切り替え可能なタグ絞り込み
- **気分ショートカット**: 「作業したい」「リラックス」など4グループから選択
- **おすすめ機能**: いいねしたカフェのタグ傾向からおすすめを表示
- **ダークモード**: ヘッダーのトグルで切替（localStorage保存）
- **レスポンシブ**: スマートフォン対応（ハンバーガーメニュー）

## 使い方

1. トップページで地図上のピンまたは右のリストからカフェを選択
2. 「詳細を見る」で写真・タグ・コメント・いいねを確認
3. ログイン後はカフェの登録・タグ追加・コメント投稿が可能
4. ヘッダーの「絞り込み」からタグでカフェをフィルタリング

## 技術スタック

- **フロントエンド**: HTML / CSS / JavaScript（Vanilla JS）
- **地図**: Leaflet + OpenStreetMap / CartoDB Positron tiles
- **バックエンド**: Supabase（PostgreSQL, Auth, Storage）
- **フォント**: Zen Maru Gothic（Google Fonts）

## 開発

```bash
git clone <repository-url>
cd cafe-map
npx serve -l 3000 .
```

`http://localhost:3000` で起動します。Supabaseの認証・ストレージを使用するため `file://` では動きません。

### seedデータの投入

`seed.html` をブラウザで開き、Supabaseの service_role key を入力すると、10件のカフェ・タグ・いいねが投入されます（要: Supabase管理画面 → Project Settings → API）。
